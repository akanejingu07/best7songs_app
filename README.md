# Best 7 Songs App

好きな楽曲を「ベスト7」として登録・閲覧できるシンプルな Web アプリです。  
楽曲名、アーティスト、URL、コメントを記録し、自分だけの音楽リストを作成できます。

## URL
https://best7songs-app.onrender.com

## 機能
- 楽曲の登録・一覧表示
- 詳細ページでの情報表示（URL クリック対応）
- 投稿の編集・削除
- レスポンシブ対応（PC・スマートフォン）

## 使用技術
- Python / Flask
- HTML / CSS
- SQLite
- Render（デプロイ）
- GitHub（バージョン管理）

## 工夫した点
- 入力項目を最小限にし、直感的に操作できる UI を意識しました  
- 詳細ページでは外部 URL を安全に遷移できるように実装しています  
- CSS を調整し、フォームやカードの視認性を高めました
